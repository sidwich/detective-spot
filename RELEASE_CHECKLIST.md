## 📋 发布前必须完成的步骤

### 第一步：注册微信小程序账号
1. 访问 [微信公众平台](https://mp.weixin.qq.com)
2. 点击"立即注册" → 选择"小程序"
3. 填写邮箱、密码完成注册
4. **关键**：服务类目必须选择 **"游戏"**
5. 完成邮箱激活和信息登记

### 第二步：填写小程序信息
1. 登录小程序后台
2. 设置 → 基本设置
3. 填写：
   - 小程序名称：找茬侦探社
   - 小程序简称：（可选）
   - 头像：上传游戏Logo
   - 介绍：一款考验观察力的找茬解谜游戏
   - 服务类目：游戏-休闲游戏

### 第三步：获取AppID
1. 开发 → 开发设置
2. 复制"AppID(小程序ID)"
3. 替换项目中的 `YOUR_APPID_HERE`：
   ```json
   // project.config.json
   {
     "appid": "wxYOUR_APPID_HERE"
   }
   ```

### 第四步：完成备案（强制要求）
1. 设置 → 基本设置 → 去备案
2. 准备材料：
   - 主办单位证件（企业营业执照/个人身份证）
   - 主体负责人身份证件
   - 小程序负责人身份证件
   - 人脸识别验证
3. 填写备案信息，提交审核
4. 等待审核（通常7-20个工作日）

### 第五步：申请软件著作权（软著）
1. 准备材料：
   - 源代码（前后各30页，共60页）
   - 操作说明书/设计文档
   - 申请表
2. 提交途径：
   - 中国版权保护中心官网
   - 或委托代理机构
3. 办理周期：1-3个月（电子版权认证更快）

### 第六步：填写游戏资质
1. 游戏 → 版本管理 → 提交审核
2. 填写游戏信息：
   - 游戏名称：找茬侦探社
   - 游戏简介：考验眼力的找茬解谜游戏，包含20个精心设计的关卡...
   - 游戏类目：休闲益智
   - 适龄提示：8+
3. 上传资质：
   - [ ] 软件著作权证书
   - [ ] 游戏自审自查报告（后台下载模板填写）
   - [ ] 版号证明（如有内购需要）
   - [ ] ICP备案截图

### 第七步：配置隐私政策
1. 将 `privacy/index.html` 部署到服务器
2. 获取隐私政策URL
3. 设置 → 基本设置 → 用户隐私保护指引
4. 填写隐私政策URL
5. 填写数据收集说明

### 第八步：上传代码
1. 下载安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目：
   - 项目目录：选择本项目根目录
   - AppID：填写你的小程序AppID
3. 点击"上传"按钮
4. 填写版本号（如：1.0.0）和项目备注

### 第九步：提交审核
1. 进入小程序后台 → 版本管理
2. 找到"开发版本"，点击"提交审核"
3. 填写审核信息：
   - 游戏测试账号：（如有账号系统）
   - 游戏测试密码：
   - 游戏内容介绍：详细描述玩法、特色
   - 游戏截图：5张以上游戏截图
   - 游戏录屏：完整玩法展示视频
4. 确认资质文件已上传
5. 提交审核

### 第十步：发布上线
1. 等待审核（通常1-3个工作日）
2. 审核通过后，微信会发送通知
3. 版本管理 → 审核版本 → 提交发布
4. 选择发布方式：
   - 全量发布：所有用户可见
   - 灰度发布：部分用户先体验

## ⚠️ 重要注意事项

### 1. 包体积限制
- 主包必须 ≤ 4MB
- 小游戏总包 ≤ 20MB
- 当前项目构建后约 380KB gzip，符合要求

### 2. 必须包含的功能
- [x] 未成年人防沉迷系统
- [x] 隐私政策入口
- [x] 用户协议（可在设置中添加）
- [x] 客服联系方式

### 3. 审核常见拒绝原因
- ❌ 游戏闪退、卡顿
- ❌ 诱导分享（如"不分享无法继续"）
- ❌ 涉黄、涉赌、涉暴内容
- ❌ 未提供隐私政策
- ❌ 侵权（图片、音乐、字体等）
- ❌ 测试内容未清理干净

### 4. 性能优化建议
- 图片资源使用WebP格式
- 大资源使用远程加载（CDN）
- 及时清理不用的纹理
- 使用对象池减少GC
